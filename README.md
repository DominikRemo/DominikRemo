## Projects

**[Artemis](https://github.com/ls1intum/Artemis):** Master's thesis work extending Practice Mode across programming, text, and modeling exercises, plus an LLM-driven workflow for generating adaptive exercise variants. Built with Spring Boot, Angular, and Spring AI. Earlier, my bachelor's thesis cut the server test suite's CI runtime from 25 to 10 minutes by parallelizing with JUnit 5, and fixed 27 intermittently failing tests to make the suite more reliable.

**[404: Trust not Found](https://github.com/DominikRemo/404-trust-not-found):** A serverless, peer-to-peer social deduction game built with Vue 3 and PeerJS, themed around software development. No backend required to play with a distributed team. I built it in my spare time to strengthen team building at Hacker School's remote-first team, based on a physical card game we grew to love at team meetups. I use Claude Code as an AI assistant on this project, reviewing and testing every generated change before it goes in.

**[Hacker School Utility Extension](https://github.com/DominikRemo/HackerSchoolUtilityChromeExtension):** Chrome/Firefox extension that lets Hacker School staff bulk-delete student projects on CodeHS instead of clicking through them one by one, saving up to 20 minutes a day and cutting the organizational overhead after each course.

**[team-closed-ai](https://github.com/AET-DevOps25/team-closed-ai):** Kanban board with a GenAI assistant that classifies user intent, answers project questions via a RAG pipeline, and creates tickets. Built the FastAPI/LangChain service and owned CI/CD with Terraform and Ansible on AWS.

**[edutelligence](https://github.com/ls1intum/edutelligence):** Research tool for comparing AI-generated feedback against human tutor feedback. We also generated ratings using LLM-as-a-Judge. Built with Next.js, React, and a Python/Jupyter study pipeline.

**[Orpheus](https://github.com/fa25genai/orpheus):** Two-week Ferienakademie project turning lecture scripts into AI-avatar lecture videos. Built the slide-generation microservice and a LangChain facade unifying multiple LLM providers.
